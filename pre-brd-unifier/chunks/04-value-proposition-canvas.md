# Value Proposition Canvas

A strategic tool used to ensure the product or service is aligned with customer needs. It has two sides: the customer profile (jobs, pains, gains) and the value map (products and services, pain relievers, gain creators). Product-market fit is achieved when the value map matches the customer profile.
