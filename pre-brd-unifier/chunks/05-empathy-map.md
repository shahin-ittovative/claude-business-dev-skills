# Empathy Map

A collaborative visualization tool used to understand users better. It captures what a user says, thinks, does, and feels, plus their pains and gains, putting the team in the user's shoes and aligning product design with real needs and desires.

Fill one table per persona. Duplicate the block below for each persona and set the persona name.

## Persona: _<persona name>_

| Section | What to capture | Answer |
|---|---|---|
| Says | What the user says out loud in interviews or feedback. |  |
| Thinks | What the user thinks but might not voice directly. |  |
| Does | What the user actually does and how they behave. |  |
| Feels | The user's emotional state, concerns, and desires. |  |
| Pains | Frustrations or challenges. |  |
| Gains | What the user hopes to achieve. |  |
