# Product Strategy Canvas

A structured visual tool that helps product teams define and align the strategic direction of a product. It captures the why, who, what, and how of the product to ensure clarity, focus, and alignment across product, design, tech, and business teams, and to support prioritization and pitching.

| Element | What to capture | Answer |
|---|---|---|
| Vision | The long-term purpose and inspiration behind the product. Why are we building it? |  |
| Target Market | The specific customer segments or personas you aim to serve. |  |
| Customer Needs | The key pain points, desires, or jobs-to-be-done for those segments. |  |
| Value Proposition | What makes the product uniquely valuable to those customers. |  |
| Differentiators | How the product stands out from competitors. |  |
| Success Metrics | How success will be measured (retention, revenue, adoption). |  |
| Channels | Where and how customers are reached (sales, marketing, support). |  |
| Revenue Model | How the product will make money. |  |
| Cost Structure | The main cost drivers (tech, support, marketing, operations). |  |
