# Lean Canvas

A one-page business plan template designed for startups and innovators. It helps map out the most critical elements of a business idea quickly and clearly. Adapted from the Business Model Canvas and optimized for early-stage products where uncertainty is high. It emphasizes real-world problems and customer segments, and focuses on problems rather than features so the team avoids building things nobody needs.

| Section | What to capture | Answer |
|---|---|---|
| Problem | Top 1 to 3 problems the product solves. |  |
| Customer Segments | Target users or personas. |  |
| Unique Value Proposition | Why the solution stands out. |  |
| Solution | Main features or services. |  |
| Channels | How users are reached. Owned (website, blog, email), paid (ads, social, influencers), partner (resellers, affiliates), sales (inside sales, field sales, product-led growth). |  |
| Revenue Streams | How the product makes money. |  |
| Cost Structure | Key costs to operate. |  |
| Key Metrics | How success is measured. |  |
| Unfair Advantage | Hard-to-copy benefits. |  |
| Key Partners | Entities or individuals who help deliver the value proposition. |  |
| Key Activities | The most important tasks to deliver value, attract users, and maintain operations. |  |
| Key Resources | The assets (human, digital, financial) needed to operate the business. |  |
| Customer Relationships | How customers are acquired, retained, and grown. |  |
